# LinuxScripts
Linux自用一键脚本，集成如IP修改、Hostname修改、MosDNS安装及Ui面板安装和Singbox安装等功能，后续随使用陆续更新，敬请关注

## 功能简介
> 1. IP
>   1) 静态IP修改
>   2) DHCP
> 2. HostName
> 3. 基础环境设置
>   1) Update & Upgrade
>   2) 安装程序依赖
>   3) 设置时区为Asia/Shanghai
>   4) 设置NTP为ntp.aliyun.com
>   5) 关闭53端口监听
>   9) 一键安装以上所有基础设置
> 4. MosDNS
>   1) 安装Mosdns
>   2) 重置Mosdns缓存
>   3) 安装Mosdns UI
>   4) 卸载Mosdns
>   8) 一键安装Mosdns及UI面板
>   9) 一键卸载Mosdns及UI面板
> 5. Sing-box
>   1) 安装官方sing-box/升级
>   2) hysteria2 回家
>   3) 卸载sing-box
>   4) 卸载hysteria2 回家
>   9) 一键卸载singbox及HY2回家
